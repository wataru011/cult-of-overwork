# 長時間労働診断 — Work Style Type Quiz

MBTI風の「長時間労働タイプ」診断サイトです。16個の質問に答えると、あなたが全16タイプのうちどの長時間労働タイプかを判定します。

## 仕組み

- 各設問は1つのタイプを象徴するステートメント（全16タイプぶん）
- 各設問に「とてもそう思う(+2)〜まったくそう思わない(-2)」の5段階で回答する
- そう思うほど、そのタイプへの加点が大きくなる
- 全設問終了後、合計得点が最も高いタイプを診断結果として表示
- 結果画面では強み・注意点・前向きなアドバイス・相性タイプを提示
- トップ画面にバージョンを表示（`v<コミット数>+<短縮SHA>`）。デプロイ時に GitHub Actions が自動で埋め込むため、コミットのたびに進む。ローカルで直接開いた場合は `dev` と表示

## 使い方

ビルド不要の静的サイトです。ローカルで開く場合:

```bash
# 任意の静的サーバーで起動（例）
python3 -m http.server 8000
# → http://localhost:8000 をブラウザで開く
```

`index.html` をブラウザで直接開いても動作します。

## ファイル構成

| ファイル | 役割 |
| --- | --- |
| `index.html` | 画面構造（スタート / 質問 / 集計 / 結果） |
| `styles.css` | スタイル・アニメーション |
| `data.js` | タイプ定義と設問データ |
| `app.js` | 診断ロジック・スコアリング・シェア |
| `cards/<id>.svg` | タイプ別のシンボル画像（結果画面に表示） |
| `ogp.png` / `ogp.svg` | OGP画像（SNSシェア用） |
| `tools/gen-cards.js` | カード・OGPを生成するスクリプト |

### 画像の再生成

シンボル画像とOGPは `tools/gen-cards.js` で生成しています。

```bash
node tools/gen-cards.js          # cards/*.svg と ogp.svg を生成
# OGPのPNG化（cairosvg を使用）
python3 -c "import cairosvg; cairosvg.svg2png(url='ogp.svg', write_to='ogp.png', output_width=1200, output_height=630)"
```
