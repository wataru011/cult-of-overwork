# 長時間労働診断 — Work Style Type Quiz

MBTI風の「長時間労働タイプ」診断サイトです。12個の質問に答えると、あなたがどの長時間労働タイプかを判定します。

## 診断できるタイプ

| タイプ | ひとこと |
| --- | --- |
| 🩹 自己犠牲タイプ | あなたが倒れたら、誰がチームを守るの？ |
| 🌏 社会奉仕タイプ | この仕事の先に、まだ見ぬ誰かの笑顔がある。 |
| 🚀 夢実現タイプ | 今の残業は、未来へのフライング。 |
| 🔥 長時間労働が好きすぎるタイプ | 働いてるときがいちばん、自分らしい。 |
| 💀 破滅タイプ | 限界？それ、とっくに超えてるかも。 |
| 🎨 自己表現タイプ | 仕事は、わたしという作品。 |

## 仕組み

- 各設問の選択肢が6タイプそれぞれにポイントを加算
- 全設問終了後、合計得点が最も高いタイプを診断結果として表示
- 結果画面では各タイプとの一致度をバーで可視化し、強み・注意点・アドバイス・相性タイプを提示

## 使い方

ビルド不要の静的サイトです。ローカルで開く場合:

```bash
# 任意の静的サーバーで起動（例）
python3 -m http.server 8000
# → http://localhost:8000 をブラウザで開く
```

`index.html` をブラウザで直接開いても動作します。

## ファイル構成

| ファイル | 役割 |
| --- | --- |
| `index.html` | 画面構造（スタート / 質問 / 集計 / 結果） |
| `styles.css` | スタイル・アニメーション |
| `data.js` | タイプ定義と設問データ |
| `app.js` | 診断ロジック・スコアリング・シェア |

## 公開（GitHub Pages）

リポジトリの Settings → Pages で、ブランチを指定して公開できます。
