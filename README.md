# 長時間労働診断 — Work Style Type Quiz

MBTI風の「長時間労働タイプ」診断サイトです。16個の質問に答えると、あなたが全16タイプのうちどの長時間労働タイプかを判定します。

## 仕組み

- 16タイプを循環配置し、各設問が6タイプ分の選択肢を提示（各タイプはちょうど6問に登場）
- 各タイプには1問だけ「シグネチャー設問(3点)」があり、一貫して選べば必ずそのタイプになる
- 全設問終了後、合計得点が最も高いタイプを診断結果として表示
- 同点トップは「シグネチャー設問(3点)を選んだタイプ」を優先し、それでも並べばよりレアなタイプを優先（MBTI流のマイノリティ優先）
- 結果画面では強み・注意点・前向きなアドバイス・相性タイプ・要注意（相性の悪い）タイプを提示
- トップ画面にバージョンを表示（`v<コミット数>+<短縮SHA>`）。デプロイ時に GitHub Actions が自動で埋め込むため、コミットのたびに進む。ローカルで直接開いた場合は `dev` と表示

## 使い方

ビルド不要の静的サイトです。ローカルで開く場合:

```bash
# 任意の静的サーバーで起動（例）
python3 -m http.server 8000
# → http://localhost:8000 をブラウザで開く
```

`index.html` をブラウザで直接開いても動作します。

## ファイル構成

| ファイル | 役割 |
| --- | --- |
| `index.html` | 画面構造（スタート / 質問 / 集計 / 結果） |
| `styles.css` | スタイル・アニメーション |
| `data.js` | タイプ定義と設問データ |
| `app.js` | 診断ロジック・スコアリング・シェア（対応ブラウザでは結果カード画像を添付） |

| `tools/gen-cards.js` | カード・OGPを生成するスクリプト |

### 画像の再生成

シンボル画像とOGPは `tools/gen-cards.js` で生成しています。

OGPはスタート画面（暗い背景＋白いカード）を模したデザインです。

```bash
node tools/gen-cards.js          # cards/*.svg と ogp.svg を生成
# OGPのPNG化（日本語フォント込みでラスタライズ。@resvg/resvg-js を使用）
npm i @resvg/resvg-js
node -e "const{Resvg}=require('@resvg/resvg-js');const fs=require('fs');const r=new Resvg(fs.readFileSync('ogp.svg','utf8'),{fitTo:{mode:'width',value:1200},font:{loadSystemFonts:true,defaultFontFamily:'IPAGothic'}});fs.writeFileSync('ogp.png',r.render().asPng());"
```

※ 日本語が描画できるフォント（例: IPAGothic）が必要です。cairosvg / rsvg-convert でも可。
